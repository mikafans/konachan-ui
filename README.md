# konachan-ui

a project by using tauri & react. (also bun, vite)

## instructions

```sh
# node modules
bun install

# run locally
bun run tauri dev 
```

## helper setup

```bash
brew tap oven-sh/bun
brew install bun

cargo install create-tauri-app --locked
cargo create-tauri-app
```
